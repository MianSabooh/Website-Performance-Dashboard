# 🌐 Website Performance Dashboard (Power BI)

![Dashboard Preview](dashboard%20website%20analysis/websiteAnalysis.png)

## 📌 Project Overview
This project presents an interactive **Website Performance Dashboard** built using **Power BI**.  
It analyzes key website metrics such as:

- Page Views  
- Session Duration  
- Bounce Rate  
- Conversion Rate  
- Traffic Sources  
- Visitor Behavior  

The goal is to help businesses understand user engagement and optimize conversion strategies.

---

## 📊 Dashboard Features

### 🔹 KPI Cards
- Average Page Views: **5.01**
- Average Session Duration: **300.02 sec**
- Average Bounce Rate: **50%**
- Average Conversion Rate: **5%**

### 🔹 Visualizations
- 📊 Donut Chart → Bounce Rate (New vs Returning Visitors)
- 📊 Donut Chart → Conversion Rate (New vs Returning Visitors)
- 📊 Bar Chart → Conversion Rate by Traffic Source
- 🗺️ Map → Conversion Rate by City
- 📋 Table → Top 100 Visitors by Conversion Rate
